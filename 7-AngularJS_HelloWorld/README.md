# 🧪 EXPERIMENT: AngularJS Hello World (ng-controller, ng-model, expressions)
To create a simple AngularJS application using ng-controller, ng-model and expressions.

🧠 CONCEPT
What is AngularJS?
AngularJS is a JavaScript framework used to build dynamic Single Page Applications (SPA).

Key Concepts:
ng-app → initializes AngularJS
ng-controller → controls logic
ng-model → binds input data
{{ }} (expressions) → display data

⚙️ SOFTWARE REQUIRED
VS Code / Fedora editor
Web Browser (Chrome)
Internet (for AngularJS CDN)

✅ OUTPUT
Initially shows:
Hello Vaishnavi

When user types:
Enter Name: Rahul
Output: Hello Rahul

Updates instantly (no refresh)

🧠 Flow
User types → ng-model updates → expression updates UI automatically

This is called Two-Way Data Binding


📚 THEORY

AngularJS is a JavaScript framework used to develop dynamic web applications.
It uses directives like ng-app, ng-controller, and ng-model to bind data between the model and view.
Expressions ({{ }}) are used to display dynamic data.

📊 OUTPUT

The AngularJS application successfully displayed dynamic data using ng-model and expressions.
User input was instantly reflected in the output without reloading the page.


🎤 VIVA QUESTIONS (VERY IMPORTANT)

❓ What is AngularJS?
A framework for building dynamic web applications.

❓ What is ng-model?
It binds input data to a variable.

❓ What is ng-controller?
It defines the scope and logic.

❓ What are expressions?
{{ }} used to display dynamic values.

❓ What is two-way data binding?
Data automatically updates in both model and view.
